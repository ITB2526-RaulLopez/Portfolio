High-Availability Server Setup

Deployment of a redundant Linux infrastructure with automated backups and load balancing.

1. Infrastructure Design & VPC

Designing the network architecture using a Virtual Private Cloud (VPC) to isolate database and web server tiers.

2. Load Balancer Configuration

Setting up Nginx as a Reverse Proxy to distribute traffic between multiple nodes, ensuring 99.9% uptime.

3. Database Replication

Implementing a Master-Slave replication system to prevent data loss in case of a server failure.

4. Automated Backup Scripts

Development of Bash scripts that perform daily encrypted backups to an off-site S3 storage bucket.

5. Monitoring & Alerts

Integration with Prometheus and Grafana to monitor CPU load and receive real-time alerts via Telegram.

Configuració del Projecte GitHub

Resum dels passos per a la creació del repositori i vinculació amb PyCharm.

1. Creació del Compte i Repositori

Crea un compte a GitHub i un nou repositori amb la nomenclatura: ITB2526-TA02-NomCognom.

2. Autenticació de Dos Factors (2FA)

És obligatori activar l'autenticació en dos passos a la configuració de seguretat de GitHub.

3. Clonar a PyCharm

Des de PyCharm, selecciona "Get from VCS" i introdueix la URL del teu repositori per crear la còpia local.

4. Fitxer README.md

Assegura't que el projecte conté un fitxer README.md explicatiu.

5. Afegir Recursos i fer Commit

Afegeix fitxers externs, fes el Commit i després el Push.

Network Security Scanner

Automated tool developed in Python to identify active hosts and open ports within a local network.

1. Environment Setup & Scapy Integration

The core of the scanner uses the Scapy library to send ARP requests and discover devices in the network range.

2. Port Scanning Logic

Implementation of a multi-threaded TCP connect scanner to check for common service ports (SSH, HTTP, FTP).

3. Vulnerability Mapping

The tool compares identified service banners against a local database of known CVE vulnerabilities.

4. Data Logging & Output

Results are automatically exported to a results.log file and displayed in a clean terminal table format.

5. GitHub Deployment

Final version pushed to GitHub following the ITB standards, including a comprehensive documentation in the README.md.
